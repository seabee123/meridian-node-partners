# Meridian Node Partners

Premium World Mobile AirNode reseller platform built with React, TypeScript, Vite, and Supabase.

## Features

- Premium enterprise-grade design
- - Multi-step order form with Supabase integration
  - - Product catalog for World Mobile AirNodes
    - - Responsive design with Tailwind CSS
     
      - ## Tech Stack
     
      - - React 18 + TypeScript
        - - Vite
          - - Tailwind CSS
            - - Supabase
             
              - ## Getting Started
             
              - npm install
              - npm run dev
